# ADR 0001: Desktop-First Floating App

## Decision

Build the MVP as a Windows desktop-first floating app.

## Rationale

The user value is fast artifact decision-making while Genshin is open. A desktop floating window supports always-on-top scanning and compact feedback better than a normal web page.

## Consequences

Future web/mobile support should reuse the TypeScript probability core, but automatic screen capture remains desktop-specific.
